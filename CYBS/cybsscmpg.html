Legacy CYBS API

SCMP (Simple Commerce Message Protocol)

SCMP (Simple Commerce Message Protocol) is a legacy name-value pair API that is supported only for existing merchants who have already implemented it. These resources are for merchants who want to download the latest SCMP API client version for their existing implementations.

Secure Acceptance Checkout API option is ideal for Web merchants who prefer to host their own order page, perhaps for greater customization, and simply need a method of transferring payment data for processing.

Status: Deprecated

Alternative Solution: Rest API
